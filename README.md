 **PicMyStep** － 用照片記錄你旅程中的每一步驚喜
==========

Google Play 下載
==========
https://play.google.com/store/apps/details?id=com.pcchang.picmystep

App簡介
==========
本軟體提供一個簡便的操作介面，用最少的步驟幫助您用照片記錄旅程中的每一步驚喜。

提供以下功能：
1. 拍照完畢後可進行照片主題的編輯與加註。
2. 自動定位所在位置，填入照片地點資訊。
3. 管理歷史照片紀錄。
4. 簡易版 Google Map 工具，提供當前位置查詢、衛星圖、交通狀況等功能。

主要技術
==========
* 啟動手機內建 Camera 程式
* 定位功能
* SQLite 資料庫
* Google Map API
